# Cambrionix CLI Reference (Unofficial)

This directory contains a community-maintained technical reference for the **Cambrionix Command Line Interface (CLI)**, generated from the [Cambrionix-CLI-User-Manual.pdf](../Cambrionix-CLI-User-Manual.pdf) (Manual version: 2025-06).

## Purpose
This documentation is an AI-friendly derivative work created to improve readability for coding agents and developers working with the Cambrionix CLI.

## Reference Files
The documentation is split into the following parts:

1. [01-introduction-and-commands-a-m.md](./01-introduction-and-commands-a-m.md): Introduction, Connection Settings, Command/Response Structure, and Commands (bd through mode).
2. [02-commands-n-z-and-deprecated.md](./02-commands-n-z-and-deprecated.md): Advanced Commands (reboot through stall) and Deprecated Methods.
3. [03-errors-and-settings.md](./03-errors-and-settings.md): Error Codes, Charging Profiles, Port Modes, LED Control, and Internal Hub Settings.
